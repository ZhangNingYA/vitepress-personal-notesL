# 渲染管线与坐标变换

最核心的是：把物体从“模型空间”一路变换到“屏幕空间”。

## 1. MVP（Model / View / Projection）

常见矩阵链：

$$
\mathbf{p}_{clip} = P\,V\,M\,\mathbf{p}_{model}
$$

其中：

- $M$：模型到世界（摆放物体）
- $V$：世界到相机（把相机当作世界的逆变换）
- $P$：投影（透视/正交）

## 2. 透视投影直觉

透视投影的关键效果是“远小近大”。用齐次坐标做除法（perspective divide）：

$$
(x,y,z,w) \mapsto (x/w,\,y/w,\,z/w)
$$

::: tip 实战提示
遇到“物体被裁剪/闪烁/深度不对”，优先检查坐标系约定（右手/左手）、裁剪空间范围与深度映射。
:::
