## 1. 引入：从“矩阵”回归到“变换”
**如何引入：**
Strang 教授开场指出，之前的课程我们一直沉浸在“矩阵（Matrices）”的世界里（如 $Ax=b$, 特征值等）。但线性代数的核心不仅仅是数字排列的矩阵，更本质的是**对向量的操作**。
* **观点：** 矩阵就像是照片（基于特定坐标系），而线性变换是那个真实的动作（独立于坐标系）。
* **目标：** 理解什么是线性变换，以及它如何与矩阵联系起来（即：所有的线性变换在选定基底后，都可以表示为矩阵）。

---

## 2. 线性变换的定义 (Definition)
**讲解逻辑：**
教授给出了判定一个变换 $T$ 是否为“线性”的两个核心条件。假设 $T$ 将空间 $V$ 中的向量映射到空间 $W$ 中（$T: V \to W$）。

### 核心条件
对于任意向量 $v, w$ 和标量 $c$：
1.  **加法性 (Additivity):** $T(v + w) = T(v) + T(w)$
2.  **齐次性 (Homogeneity):** $T(cv) = cT(v)$

**合并为一条规则：**
$$T(cv + dw) = cT(v) + dT(w)$$
*这就叫线性组合的变换等于变换的线性组合。*

### 重要性质（用于快速排除非线性）
* **零向量法则：** 线性变换必须将原点映射到原点。
    $$T(0) = 0$$
    *推导：* 取 $c=0$，则 $T(0 \cdot v) = 0 \cdot T(v) = 0$。

## 3. 例子与反例 (Examples & Non-Examples)
Strang 通过几何直观和简单的数学操作来举例。

### A. 线性变换的例子
1.  **投影 (Projection):**
    * *动作：* 将平面上的向量投影到一条直线上。
    * *直观：* 它是线性的，因为两个向量和的投影等于它们各自投影的和。
2.  **旋转 (Rotation):**
    * *动作：* 将平面上的向量绕原点旋转 $45^\circ$。
    * *直观：* 整个平面一起转，网格线保持平行且等距，原点不动。
3.  **矩阵乘法 (Matrix Multiplication):**
    * *动作：* $T(x) = Ax$。
    * *结论：* 这是最常见的线性变换形式。实际上，这一讲的结论就是**所有线性变换最终都能写成这种形式**。

### B. 非线性变换的例子 (Counter-Examples)
1.  **平移 (Shift/Translation):**
    * *定义：* $T(v) = v + v_0$ (其中 $v_0 \neq 0$)。
    * *为什么不是线性？* 它不仅破坏了加法性，最明显的是它移动了原点：$T(0) = v_0 \neq 0$。
2.  **取模/长度 (Magnitude):**
    * *定义：* $T(v) = \|v\|$。
    * *为什么不是线性？* 输入是向量，输出是数字（这没问题），但 $T(2v) = 2\|v\|$ 虽然满足乘法，却不满足加法（三角形两边之和大于第三边，不一定相等）。
    * *修正：* 即使 $T(v) = v^2$ (分量平方) 也不是线性的，因为 $(2v)^2 = 4v^2 \neq 2v^2$。

## 4. 核心概念：线性变换与矩阵的桥梁
**如何讲解：**
这是本节课的重头戏。Strang 提出了一个问题：**如果不给我坐标，只描述变换（如“旋转45度”），我怎么计算？**
为了计算，我们需要将“抽象变换”转化为“具体矩阵”。

**口诀：** "No coordinates, no matrix." (没有坐标就没有矩阵)。
**关键步骤：** 必须为输入空间 $V$ 和输出空间 $W$ 选定**基 (Basis)**。

### 构造矩阵 $A$ 的规则 (The Rule)
假设我们要构造一个矩阵 $A$ 来代表变换 $T$。
1.  假设输入空间 $V$ 有基 $v_1, v_2, ..., v_n$。
2.  假设输出空间 $W$ 有基 $w_1, w_2, ..., w_m$。
3.  **矩阵 $A$ 的第 $j$ 列由 $T(v_j)$ 决定。**

**具体操作：**
将变换 $T$ 作用于第 $j$ 个输入基向量 $v_j$，得到的也是一个向量（在 $W$ 中）。将这个结果表示为输出基 $w$ 的线性组合：
$$T(v_j) = a_{1j}w_1 + a_{2j}w_2 + ... + a_{mj}w_m$$
这其中的系数 $(a_{1j}, a_{2j}, ..., a_{mj})$ 就是矩阵 $A$ 的**第 $j$ 列**。

> **Strang 的总结：**
> 只要知道基向量变换后去了哪里，你就知道了整个变换的一切（因为任何向量都是基的线性组合）。

## 5. 精彩实例：求导数 (Derivative) 作为线性变换
Strang 选择了一个函数空间的例子，展示了线性代数如何跨越到微积分。

### 设定
* **输入空间 $V$：** 三次多项式 $P_3$ (最高次为3，如 $c_1 + c_2x + c_3x^2 + c_4x^3$)。
    * 基 $v$：$1, x, x^2, x^3$ (维数 $n=4$)。
* **输出空间 $W$：** 二次多项式 $P_2$ (求导后次数降低)。
    * 基 $w$：$1, x, x^2$ (维数 $m=3$)。
* **变换 $T$：** 求导操作 $D = \frac{d}{dx}$。

### 构造求导矩阵 $A$
我们需要找到一个 $3 \times 4$ 的矩阵。按照“规则”逐列计算：

1.  **第一列 (处理 $v_1=1$)：**
    * $T(1) = \frac{d}{dx}(1) = 0$。
    * 用 $W$ 的基表示：$0 = \mathbf{0}(1) + \mathbf{0}(x) + \mathbf{0}(x^2)$。
    * 所以第一列是 $\begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix}$。

2.  **第二列 (处理 $v_2=x$)：**
    * $T(x) = \frac{d}{dx}(x) = 1$。
    * 用 $W$ 的基表示：$1 = \mathbf{1}(1) + \mathbf{0}(x) + \mathbf{0}(x^2)$。
    * 所以第二列是 $\begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}$。

3.  **第三列 (处理 $v_3=x^2$)：**
    * $T(x^2) = 2x$。
    * 用 $W$ 的基表示：$2x = \mathbf{0}(1) + \mathbf{2}(x) + \mathbf{0}(x^2)$。
    * 所以第三列是 $\begin{bmatrix} 0 \\ 2 \\ 0 \end{bmatrix}$。

4.  **第四列 (处理 $v_4=x^3$)：**
    * $T(x^3) = 3x^2$。
    * 所以第四列是 $\begin{bmatrix} 0 \\ 0 \\ 3 \end{bmatrix}$。

### 最终矩阵
求导操作对应的矩阵 $A_{diff}$ 为：
$$A = \begin{bmatrix} 0 & 1 & 0 & 0 \\ 0 & 0 & 2 & 0 \\ 0 & 0 & 0 & 3 \end{bmatrix}$$

**验证：**
如果输入向量是 $x^3 + x^2$ (系数向量为 $\begin{bmatrix} 0 \\ 0 \\ 1 \\ 1 \end{bmatrix}$)。
$$A \begin{bmatrix} 0 \\ 0 \\ 1 \\ 1 \end{bmatrix} = \begin{bmatrix} 0 \\ 2 \\ 3 \end{bmatrix}$$
结果对应输出基的系数：$0(1) + 2(x) + 3(x^2) = 2x + 3x^2$。这正是 $(x^3+x^2)$ 的导数！

## 6. 总结 (Summary)
1.  **线性变换**是独立于坐标系的抽象操作，满足加法性和齐次性。
2.  **矩阵**是线性变换在特定**基 (Basis)** 下的具体表达。
3.  **构造矩阵的方法**：将变换作用于每一个输入基向量，并将结果写成输出基向量的线性组合，提取系数作为矩阵的列。
4.  矩阵乘法 $Ax$ 的本质就是：$A$ 告诉我们在给定基底下，输入向量的系数如何转化为输出向量的系数。
